Trabalho de Engenharia de Software 2025.1 e Atividade Extensionista Desenvolvimento de Artefato Tecnológico

O projeto consiste na elaboração de um software seguindo as principais etapas de desenvolvimento de Software, atreladas a disciplina principal com assuntos abordados em aula.

Foi desenvolvido em um grupo de 6 pessoas, no qual estou incluído.

As etapas do trabalho são as seguintes:
– Etapa 1: Definição do processo de desenvolvimento de software e ciclo de vida
– Etapa 2: Elicitação e documentação de requisitos
– Etapa 3: Elaboração de casos de uso
– Etapa 4: Inspeção de requisitos
– Etapa 5: Planejamento de testes
– Etapa 6: Elaboração de protótipo de baixa
fidelidade (somente para a atividade extensionista)

Cada pasta referente as etapas contém um README com a breve explicação da etapa e quais documentos foram gerados. O conteúdo do trabalho está nos documentos.
